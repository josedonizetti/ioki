# Ioki

Scheme compiler based on "An Incremental Approach to Compiler Construction" paper by Abdulaziz Ghuloum

## Installation

Add this line to your application's Gemfile:

    gem 'ioki'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ioki
